## node官网
https://nodejs.org/en/

## 下载地址
https://nodejs.org/dist/v6.11.3/node-v6.11.3.tar.gz

## 解压并安装node
``` bash
[root@bogon ~]# tar xvf node-v6.11.3.tar.gz
[root@bogon ~]# cd node-v6.11.3
[root@bogon ~]# ./configure
[root@bogon ~]# make
[root@bogon ~]# make install
```
